# Volume-Control-Using-OpenCV
Increasing or Decreasing the Volume using Camera

Install Dependencies:

1) sudo apt install pulseaudio-utils
2) pip install opencv-python
3) pip install mediapipe
4) pip install numpy

If you are using the code without ros2 in windows, use pycaw instead of subprocess module.

Note: The camera will not turn off when the 'close' icon is pressed. Use ctrl+c to stop the node from the terminal
